La Redonda. Una tienda online de camisetas

Proyecto con interfaz gráfica de una tienda online de camisetas inspirada en los mejores equipos del fútbol actual.

Realizado en Eclipse, con el lenguaje de programación Java y conectado a una base de datos SQL realizada en MySQL.

Github

Juego José Modo. Un juego de aventuras

Proyecto con interfaz gráfica e hilos de un juego de aventuras. El objetivo principal del juego es conseguir eliminar al enemigo final mientras interactúas con los diversos objetos del juego, descubriendo así todas las funcionalidades.

Realizado en Eclipse, con el lenguaje de programación Java.

Github

Página José Modo. Página web del juego José Modo

Proyecto de una página web sobre el videojuego José Modo. Donde puedes encontrar toda la información, historia, así como una guía del juego.

Realizado en VSCode, usando HTML, CSS y JavaScript.
